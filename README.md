# 蟬鳴Charming | 互聯網不會遺忘

## Environment

`Node >= 6`

## Start

 - Clone
 - Install dependencies

``` bash
yarn
```

## Develop

``` bash
# serve with hot reload at localhost:8010
npm run dev
```

## Build

``` bash
# build for production with minification
npm run build
```
